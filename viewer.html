관리자 로그인 비밀번호 변경

viewer.html 로그인할 때 사용하는 비밀번호입니다.

개인키(privatekey.enc) 비밀번호 변경

기존 개인키를 새 비밀번호로 다시 암호화한 새 파일을 다운로드합니다. 모두 브라우저 안에서만 처리됩니다.

새 파일을 받으면 기존 privatekey.enc는 안전하게 폐기 또는 백업 교체. 공개키(publickey.pem)는 그대로 사용 가능 (키쌍 자체는 변하지 않음).

개인키 잠금 해제

privatekey.enc 파일과 비밀번호로 개인키를 메모리에 로드합니다. 한 번 로드하면 이번 세션 동안 다른 항목도 즉시 복호화됩니다.
