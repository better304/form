🔐 세무사 키 생성 도구

더나은세무회계 · 종단간 암호화(E2EE) 키 쌍 생성기

⚠ 매우 중요
• 이 페이지는 세무사 본인 PC에서만 실행하세요. 인터넷이 끊겨 있어도 동작합니다.
• 다운로드된 privatekey.enc와 비밀번호는 USB 등 별도 매체에 백업하세요.
• 분실 시 의뢰자가 보낸 모든 데이터를 복구할 방법이 없습니다.
• 페이지를 새로고침하거나 닫으면 메모리에서 키가 즉시 삭제됩니다.

1키 쌍 생성

브라우저 내에서 RSA-OAEP 4096비트 키 쌍을 만듭니다. 약 5~15초 걸립니다.

키 생성 중… 페이지를 닫지 마세요.

2공개키 (publickey.pem)

이 파일은 의뢰자 폼 사이트에 공개됩니다. 의뢰자 브라우저가 민감정보를 암호화할 때 사용합니다.

3개인키 보호 비밀번호 설정

개인키 파일은 비밀번호로 한 번 더 암호화됩니다. 이 비밀번호도 잊지 마세요.

아직 입력 전

4개인키 다운로드 (privatekey.enc)

이 파일과 비밀번호는 세무사 PC + USB 백업에만 보관하세요.

자기검증 결과: 대기 중

✅ 키 생성 완료

  1. publickey.pem → 폼 사이트 (better304.github.io/form/publickey.pem) 위치에 업로드
  2. privatekey.enc + 비밀번호 → USB 등 안전한 매체에 백업 (최소 2벌 권장)
  3. 비밀번호는 종이에 적어 별도 보관(브라우저·메모장 저장 금지)
  4. 이 페이지를 닫으면 모든 키가 메모리에서 삭제됩니다
  5. 키를 새로 생성하면 기존 의뢰자 데이터는 모두 복호화 불가해지므로 주의
